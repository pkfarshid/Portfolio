## 🚀 Quick Start

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional installations required

### How to Use

#### Main Portfolio:
1. Navigate to `index.html` or visit your GitHub Pages URL
2. Scroll through sections using the navigation menu
3. Click "View Experience" to jump to work history
4. Click "Get in Touch" to send an email

#### Digital Clock:
1. Open `clock.html` or navigate to the clock page
2. View 6 default time zones with real-time updates
3. Click **"+ Add Time Zone"** to add more timezones
4. Click **"Toggle 12/24 Hour"** to switch time formats
5. Click **"Reset to Default"** to restore original timezones

## 🎨 Design & Technology

### Frontend Stack
- **HTML5**: Semantic markup
- **CSS3**: 
  - CSS Grid & Flexbox for responsive layouts
  - CSS Animations (typewriter, pulse, slide effects)
  - CSS Variables for theming
  - Smooth transitions & hover effects
- **JavaScript (Vanilla)**:
  - Intersection Observer API for scroll animations
  - Intl.DateTimeFormat for timezone conversion
  - Dynamic DOM manipulation
  - Real-time clock updates using setInterval

### Design Highlights
- **Color Scheme**:
  - Primary: Cyan (#00D4FF) – Tech-focused
  - Secondary: Dark Blue (#0A0F1E) – Modern & professional
  - Accent Red (#FF3B5C) – For alerts/attention
- **Typography**: 
  - Headers: Space Grotesk (bold, geometric)
  - Body: Inter (clean, readable)
  - Code: Space Mono (monospaced)
- **Effects**:
  - Animated gradient backgrounds
  - Glowing text-shadows
  - Smooth parallax scrolling
  - Pulsing indicators
